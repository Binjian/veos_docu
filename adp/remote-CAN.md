## 可配置范围
可配置范围为torque表全表;
观测量可选取时间范围为1s-20s;

## 可选择数据范围
以下数据为可采集系统状态与范围

| 名称     | 可选取范围 | 
|----------|------------|
| 油门踏板 | 0%-100%    |   
| 刹车踏板 | 0%-100%    |   
| 电压     | 550V-700V  |   
| 电流     | >0A        |   
| 车速     | 0-100km/h  |   
| 档位     | N,P,R,D    |   

其中电压与电流均为载具放电情况下的取值范围，因充电状态下取值范围与本项目无关。

## 对接接口
接口为python文件类型，可直接调用;
实际类型为内嵌于算法的模块;
具体服务与大数据对接，实现自定义微服务器,用于安全隔离;
